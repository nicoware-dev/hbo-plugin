# Composio CLI Bridge

MVP external bridge skill: `plugin/hbo-plugin/skills/composio/SKILL.md` (`hbo-plugin:composio`)

## Scope

- Install and authenticate Composio CLI
- Search, link, and execute tools
- Fallback to `local-demo` when not configured

## Installation

Platform-specific install (Windows WSL, Linux, macOS): **`docs/COMPOSIO_CLI_INSTALL.md`**

Public docs site: [Composio CLI](/docs/composio-cli) (`apps/docs/docs/composio-cli.md`)

## Not in MVP

- WithOne / One CLI bridge
- n8n bridge
- Hard dependency on Composio for demo run

See also: `docs/TECHNICAL_SPEC.md` section 11.2.
