# Demo Script

Recommended demo flow for hackathon / video:

1. Show docs landing page (`pnpm dev:docs`)
2. Copy install prompt from `INSTALLATION_PROMPT.md`
3. Hermes installs plugin and profiles
4. Open Hermes dashboard → **Business Ops** tab
5. Show three agent cards
6. Run **Daily Ops Briefing** workflow
7. Show pending actions queue
8. Approve one action
9. Show audit event
10. Show **Tool Bridges** — Composio CLI as path to real tools
