# sales-ops

Inbound sales domain skill bundled with HBO Plugin.

## Requirements

- HBO Plugin enabled
- Demo or live workspace with leads and conversations

## Register

Loaded as `hbo-plugin:sales-ops` when plugin is enabled.

## Quick use

Detect signals → run `inbound_sales` → propose actions → user approves → execute.
