# growth-ops

Outbound growth domain skill bundled with HBO Plugin.

## Requirements

- HBO Plugin enabled
- Demo or live workspace with leads data

## Register

Ships with plugin — loaded as `hbo-plugin:growth-ops` when plugin is enabled.

## Quick use

Score leads → run `outbound_growth` → list pending actions → user approves.

## Files

- `SKILL.md` — agent instructions
- `examples/` — scenarios
- `evals/` — verification
- `skillopt-benchmark.jsonl` — optimizer tasks
