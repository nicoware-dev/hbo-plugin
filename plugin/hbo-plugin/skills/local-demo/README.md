# local-demo

Run HBO Business Ops Demo without external credentials.

## Requirements

- HBO Plugin installed and enabled
- Hermes dashboard available

## Register

Loaded as `hbo-plugin:local-demo` when plugin is enabled.

## Quick use

`hbo_load_demo_data` → `hbo_get_workspace` → run workflow → approve in dashboard.
