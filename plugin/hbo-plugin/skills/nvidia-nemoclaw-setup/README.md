# nvidia-nemoclaw-setup

Deploy HBO Plugin in NemoHermes / NemoClaw sandbox with Nemotron inference.

## Requirements

- NemoHermes / NemoClaw tooling
- `deploy/nemoclaw/Dockerfile` in repo

## Install

Skill only — full plugin ships via custom image build, not `skill install`.

## Quick use

Build image → `nemohermes onboard` → verify Business Ops tab → configure inference.

- `SKILL.md`, `examples/`, `evals/`, `skillopt-benchmark.jsonl`
- `docs/NVIDIA_NEMOCLAW.md`, `deploy/nemoclaw/policy-notes.md`
