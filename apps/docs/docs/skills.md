# Bundled Skills

| Skill | Purpose |
|-------|---------|
| `sales-ops` | Inbound sales workflow guidance |
| `growth-ops` | Outbound growth workflow guidance |
| `ops-lead` | Daily briefing and coordination |
| `local-demo` | Run demo without external credentials |
| `composio-cli` | Optional bridge to external tools |

Skills live in `plugin/hbo-plugin/skills/` and are registered by the plugin.
