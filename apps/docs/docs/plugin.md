# Hermes Plugin

The plugin registers HBO business operations tools (`hbo_*`) and bundled skills.

## Key tools

- `hbo_get_workspace` ‚Äî workspace summary
- `hbo_list_agents` ‚Äî agent profiles
- `hbo_run_workflow` ‚Äî run demo workflows
- `hbo_list_actions` / `hbo_approve_action` / `hbo_reject_action` ‚Äî action queue
- `hbo_generate_briefing` ‚Äî daily ops briefing
- `hbo_load_demo_data` ‚Äî reset demo state

Package location: `plugin/hbo-plugin/`
