# Profile Distributions

Three Hermes profile distributions ship with HBO Plugin:

| Profile | Focus |
|---------|-------|
| **Sales Ops Agent** | Inbound sales, bot QA, follow-ups |
| **Growth Agent** | Outbound leads, scoring, outreach |
| **Ops Lead Agent** | Daily briefing, prioritization, approvals |

Install from `profiles/` in this repository. Each profile includes `distribution.yaml`, `SOUL.md`, `config.yaml`, and optional `skills/` and `cron/`.
