# Business Ops Demo

A generic commerce operations workspace with bundled JSON demo data:

- Leads and prospects
- Customer conversations
- Pending action proposals
- Audit events
- Daily briefing examples

Data files: `plugin/hbo-plugin/data/business-ops-demo/`

No external credentials required. Composio can enhance the demo when configured.
