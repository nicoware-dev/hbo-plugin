# Business Ops Dashboard

One top-level **Business Ops** tab with internal pages:

| Page | Purpose |
|------|---------|
| Overview | Workspace summary cards |
| Agents | Three specialized agent profiles |
| Workflows | Run inbound sales, outbound growth, daily briefing |
| Leads | Demo leads and prospects |
| Actions | Approve/reject pending actions |
| Audit | Traceability log |
| Tool Bridges | Local demo + Composio CLI |
| Setup | Installation checklist |

Built as a Hermes dashboard extension (IIFE bundle + FastAPI routes).
