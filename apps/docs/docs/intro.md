# HBO Plugin

**Hermes Business Operations Plugin** turns Hermes into a practical operating system for commerce and business workflows.

## What it adds to Hermes

- A Hermes plugin with business ops tools and local demo state
- A **Business Ops** dashboard tab with internal pages
- Three agent profile distributions (Sales Ops, Growth, Ops Lead)
- Bundled workflow and bridge skills
- A self-contained **Business Ops Demo** workspace
